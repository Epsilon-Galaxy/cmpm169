Experiment 6 - Grammar / Haiku Generator

Description

I wanted to test out and use the tracery library for myself. I ended up creating a pretty meh haiku generator, but it's pretty fun to see what it makes. I want to try to make it more legible and sound more human like and I'll probably have to tone down or add more strict rules.

Technical

It works by using the tracery library and using the idea of replacement sort of like playing those madlib word games.

Reflection

I felt like I can do more with this. I want to be able to read what is written and perhaps generate some 3d effect when certain words are picked. Maybe make a haiku generator that is revolved around weather and depending on the conditions that are laid out in the haiku different weather effects will be played out.
